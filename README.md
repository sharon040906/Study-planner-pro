# Study-planner-pro
Study Planner Pro is a simple and interactive web application that helps students organize their academic tasks, manage deadlines, track progress, and stay productive.

The project was built using HTML, CSS, and JavaScript as a learning project to practice frontend web development concepts such as DOM manipulation, event handling, local storage, and basic authentication.

---

## 🚀 Features

### User Authentication

* User Registration
* User Login
* Login Validation using Local Storage
* Protected Planner Page
* Logout Functionality

### Task Management

* Add New Tasks
* Delete Tasks
* Mark Tasks as Completed
* Move Completed Tasks to a Separate Section
* Set Task Deadlines

### Categories

* Predefined Categories:

  * College
  * DSA
  * GATE
  * Skills
  * Projects
* Create Custom Categories

### Progress Tracking

* Total Tasks Counter
* Completed Tasks Counter
* Automatic Completion Percentage Calculation
